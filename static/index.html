How this was generated

1,000 app keyword ideas were generated across 40+ domains (health, finance, productivity, food, parenting, pets, home, automotive, mental health, education, social, travel, sports, creative hobbies, real estate, business, niche professional, seniors, sustainability, dating, weather, legal, faith, trades, outdoor adventure, music, cooking, student life, elderly care, pregnancy, financial planning, home improvement, fashion, language, games, events, transportation, wellness, writing, community, science, and more).

Deep mode (default): 1100+ keywords re-searched without the "app" suffix (people don't type "photos app" in the App Store), with 25 top results each. Re-ranked by Vibe-ROI — an LTV × demand × supply_room score. Category → ARPU → payback math is baked in, not eyeballed.

Relevance filter: each top-10 result is scored against the keyword's domain tokens. Demand / Gini / weak-app counts are recomputed using only relevant apps, so fuzzy matches like "jury duty tracker" → USCIS case trackers can't inflate the signal. Keywords whose top-10 is <30% relevant get verdict OFF-INTENT; 30–50% get a weak-intent penalty.

Verdicts map from Vibe-ROI: GREAT (≥8), GOOD (≥4), COMPETITIVE (≥1.5), NICHE (≥0.5), AVOID (below), OFF-INTENT (App Store returns fuzzy matches). Click column headers to sort; click a keyword to run the analyzer.

Keyword clusters — grouped by theme · category

Two theme modes: 2-word (default) groups keywords by a bigram of two shared domain tokens — e.g. #baby + milestone, #noise + white, #blood + pressure. 1-word mode groups by a single token and splits by App Store category (car · Finance separates from car · Utilities).

Bigrams need at least 2 keywords sharing the pair to cluster; a one-off like "car rental compare" stays as a singleton unless another keyword contains both car and rental. Top apps and rating volume are aggregated across all members for real competitive footprint; best Vibe-ROI flags whether any angle is worth attacking.

Picks, ranked for vibe-code + paid-ads

Each pick scores Money Score (LTV potential, 0-100), Market Opportunity, and Build Complexity (1-10, lower = faster to ship). The Vibe-ROI score combines all three with the paid-ads math below: money_score × (11 − complexity) / payback_months.

Payback months assume the cheaper of Meta/Google CPI, a 5% install→paying conversion, and midpoint ARPU. Use the sort dropdown to re-rank. Click any card to expand competitors, pain points, and build breakdown.
